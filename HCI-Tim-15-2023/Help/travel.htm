Search

Searches the travels by name. Typing inside the search field automatically updates the table.

Add

The add button opens the create dialog

Filter settings

Allows the user to filter the table by cost and number of locations. All inputs need to be valid numbers. Complete the filtering and update the table by clicking the Filter button

Filter button

Allows the user to filter the table by cost and number of locations. Updates the table using the criteria in filter settings.

Table

The table displays the id, name,cost, list of locations of travels. You can sort by any given value by clicking on the column header. Clicking on an item in the table displays the full values of a travel in the section below.

Selected Display

The fields display the full information about a travel. Selecting different items in the table changes the values of the fields. You can scroll through the list of locations.

Edit

The edit button opens the edit dialog

Delete

Deletes the selected travel. A travel has to be selected in the table first in order for the button to work.

Name

Name of the new location. Cannot be left empty.

Address

Address of the new location. The input has to be a valid address, if it's not you will receive a warning and the location won't be created.

Cost

Cost of the new location. The input has to be a valid number.

OK Button

Adds the new location and closes the dialog. Checks the input fields, if they are not valid an error will be issued.

Cancel button

Closes the dialog and returns to the table overview page.

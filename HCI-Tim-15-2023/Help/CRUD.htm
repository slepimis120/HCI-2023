Search

Search is enabled on both the name and the address of a location. Typing inside the search field automatically updates the table.

Add

The add button opens the create dialog

Table

The table displays the id, name, address and cost of locations. You can sort by any given value by clicking on the column header. Clicking on an item in the table displays the full values of a location in the section below.

Selected Display

The fields display the full information about a location. Selecting different items in the table changes the values of the fields. By clicking the edit button, it enables the input fields.

Edit

Allows the editing of the input fields in the before mentioned section. The fields cannot be left empty. The cost field has to be a valid number. Changes to the cancel button when clicked.

Confirm changes by clicking the confirm button. Or cancel them by clicking the cancel button.

Confirm Edit Button

Confirms the changes made and updates the table.

Delete

Deletes the selected location. A location has to be selected in the table first in order for the button to work.
